# Branch Prediction

The CPU is trying hard to execute instructions as fast as it can. Branch Prediction unit purpose is to guess which 
way execution of logical condition is going to go to, to "if" branch or to "else" branch, before condition result is 
calculated completely.

Making predictions trivial will minimize prediction errors and force Branch Predictor to guess correctly most of 
the time.
